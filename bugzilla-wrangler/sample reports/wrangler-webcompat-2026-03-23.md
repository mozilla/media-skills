### Theme 1 — UA Sniffing / "Firefox Not Supported" Epidemic

**User-facing impact:** Users land on hundreds of websites — spanning banking, healthcare, government, travel, gaming, and e-commerce — and are told Firefox is unsupported. In most cases the page is fully functional but the site's user agent detection logic refuses to serve it to Firefox. In worse cases, the site actively blocks or degrades the experience with no workaround available to the user.

**Root cause:** Sites are performing UA string checks rather than feature detection. Two massive Knowledge Base tracking bugs catalogue the problem. A sub-cluster is driven specifically by Akamai CDN's bot/browser detection logic, which has been increasingly flagging Firefox (especially Nightly) as a blocked client.

### Suggestions for Improving This Skill

1. **Akamai / CDN cluster detection** — The Akamai blocking pattern (1903597) was only discoverable via dep crawl. Consider adding a dedicated keyword expansion step that searches for terms like "Akamai", "blocked by CDN", or "403" in webcompat components to surface CDN-blocking sub-clusters without requiring dep graph traversal.

2. **Auto-surface "REOPENED" bugs** — Several high-signal bugs were REOPENED (rte.ie, port8.fi, YouTube navigation, GeForce Now). REOPENED status with >2 reopens is a strong signal that the root fix is unstable; a scoring bonus for multiple-reopen count would surface these.

3. **Assignee load heuristic** — twisniewski is carrying an enormous fraction of the UA/sitepatch workload (dozens of ASSIGNED bugs). Adding a "per-assignee workload concentration" metric to the report would help identify staffing pressure on the webcompat team.

4. **Site-patch longevity tracking** — Many bugs have `webcompat:sitepatch-applied` and `leave-open`. There is no current heuristic for how long a sitepatch has been in place without the underlying bug being fixed. A "sitepatch age" dimension (creation_time of the bug to today, when sitepatch-applied is set) would identify interventions that have become de-facto permanent.
